Devflo란?

자세히 보러가기

코딩이란?

컴퓨터 언어를 이용하여 컴퓨터와 대화하는 것

개발(開發, 영어:development)은 무엇인가를 보다 쓸모 있거나 향상된 상태로 변화시키는 행위이다. 소프트웨어 개발(영어:software development)은 시장 목표나 사용자의 요구를 소프트웨어 제품으로 만드는 과정이다.

개발자란?

프로그램 만드는 일을 하는, 혹은 그게 직업인 사람

개발자 종류

웹 프로그래머
웹 개발자는 HTTP 프로토콜을 커뮤니케이션 매체로 사용하는 웹 페이지, 웹 사이트 등 WWW 기반 소프트웨어 개발자 또는 소프트웨어 엔지니어를 말한다.
응용프로그래머
응용프로그램이 동작하도록 시스템을 관리하는 엔지니어를 말한다.
시스템 프로그래머
안드로이드, iOS, Window, mac os x에서 동작하는 애플리케이션을 개발하는 엔지니어를 말한다. 응용프로그램이 동작하도록 시스템을 관리하는 엔지니어를 말한다.
DBA, 데이터베이스
데이터베이스를 관리하는 엔지니어를 말한다.
네트워크.서버.보안
PM
해당 프로젝트를 지휘, 관리하며 프로젝트를 이끄는 사람.
퍼블리싱
HTML 중심이거나, 서버사이드가 감싸는 웹 구조의 형태를 지향하는 업무 스타일의 직군으로 서 웹 퍼블리셔는 사용자에게 보여지는 인터페이스 영역을 작업하는 엔지니어를 말한다. 웹 퍼블리셔는 해외에서는 UI 개발자로 불린다.
게임
컴퓨터 게임이나 비디오 게임을 만드는 엔지니어를 말한다.
ERP.시스템분석.설계
빅데이터
빅데이터를 분석하고 해석하는 엔지니어
AI
컴퓨터와 로봇이 인간처럼 생각하고 행동하도록 기술을 개발하는 엔지니어를 말한다.

‘Devflo’의 목적

한 번에 개발자에 대한 정보를 얻을 수 있는 통로가 되기 위해서 만든 사이트입니다.
대학교에서 이론적인 것은 배우나, 개발자에 대해 알 수가 없었습니다.
개발자가 되려고 정보를 찾아봤는데, 한 번에 볼 수 있는 사이트가 없었습니다.
** 비영리적인 목적으로 만들었습니다 **
